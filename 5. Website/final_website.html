About This Project

How It Was Built

A complete end-to-end data science pipeline built for the MCA Statistical Analysis & Visualization course.

This project scrapes, cleans, and analyzes Amazon India's smartphone listings using Python. Starting from raw HTML, it produces a clean dataset of 399 smartphones across 32 brands — then applies statistical methods to extract meaningful market insights.

Key findings: Redmi dominates review volume, OnePlus leads in ratings, Apple sits alone in the flagship premium, and mid-range (₹15K–₹30K) is the largest segment by listing count.

Python 3BeautifulSoup4PandasNumPy SQLiteJupyter NotebookPower BI Chart.jsHTML / CSS / JS
1
Web Scraping
Custom Python scraper with BeautifulSoup extracted 488 raw listings from Amazon India across multiple price-range pages.
2
Data Cleaning
Pandas pipeline: removed BOM chars, filled missing ratings (median), zeroed null discounts, filtered accessories (price ≤ ₹5K), computed original_price from discount.
3
Statistical Analysis
Brand benchmarking, price segmentation, correlation analysis (price vs rating), discount strategy mapping, and composite value scoring.
4
Visualization
Power BI dashboards for business reporting + this interactive website with live filtering, comparison, and chart exploration.
Built as part of MCA — Statistical Analysis & Visualization for Marketing · Amazon India Dataset · 2025 · Data: final_cleaned.csv (399 records)
